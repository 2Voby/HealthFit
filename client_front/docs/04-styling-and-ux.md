# Стилізація та UX

## Styling stack

- Tailwind CSS із CSS-змінними у `src/index.css`.
- Конфіг кольорів і radius розширений у `tailwind.config.js`.
- Схема Shadcn: `components.json` (`style: new-york`, `baseColor: neutral`).

## Поточні візуальні патерни

- Основний UI-контекст: зелена палітра (`#1a7a4a`, `#173325`, м'які зелені фони).
- Картки: заокруглення 20-30px, тонкі межі, м'які тіні.
- Типографіка:
  - Великі заголовки екранів (`text-[30px..40px]`).
  - Підписи та helper-тексти `text-[11px..15px]`.
- Іконки: Lucide + емодзі (на головній).

## Адаптивність

- Mobile-first верстка в межах `max-w-[430px]`.
- Фіксований нижній action-блок у квізі з урахуванням `env(safe-area-inset-bottom)`.
- Горизонтальні списки на екрані результату використовують `snap-x` для карток.

## Інтеракції

- Валідація кроку перед переходом далі (`isQuestionAnswered`).
- Помилки кроків:
  - toast (`react-hot-toast`),
  - локальна підсвітка поля/карток.
- Для numeric-питань доступні:
  - ручне введення,
  - scroll-wheel,
  - кнопки збільшення/зменшення.

## Рекомендації для розвитку UI

- Нові reusable-елементи додавати в `src/components/ui`.
- Feature-специфічні компоненти залишати поруч зі сторінкою (`src/pages/.../components`).
- Динамічний текст для кроку задавати через `infoTitle/infoDescription` (string або function).
