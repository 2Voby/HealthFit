# UI Документація

Цей розділ описує поточну реалізацію клієнтського інтерфейсу.

## Зміст

- [01-ui-overview.md](./01-ui-overview.md) - архітектура UI, роутинг і базовий флоу.
- [02-pages.md](./02-pages.md) - опис екранів та їхньої поведінки.
- [03-components.md](./03-components.md) - спільні й feature-компоненти.
- [04-styling-and-ux.md](./04-styling-and-ux.md) - дизайн-токени, Tailwind-патерни та UX-рішення.

## Швидкий старт для перевірки UI

```bash
npm run dev
```

Відкрий `http://localhost:5173` і пройди шлях:
`/` -> `/quiz` -> `/result`.
